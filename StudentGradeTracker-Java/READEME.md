# Student Grades Project

## Description
This project allows a teacher to enter students' grades, compute their average, highest, and lowest scores, and generate a report in CSV format.

## Features
- Input student information and grades
- Compute average, highest, and lowest grades
- Generate a CSV report with student statistics

## Setup
1. Navigate to the `src` directory.
2. Compile the Java files:
    ```bash
    javac *.java
    ```
3. Run the `Main` class:
    ```bash
    java Main
    ```

## Report
The generated report will be saved as `data/grades.csv`.
